# Set up your environment

## Install VsCode
```
brew install --cask visual-studio-code
```

### Install Extensions in VsCode
Look at the left sidebar, click on "Extensions" tab and install:
- Python
- Pylance
- Jupyter
- GitLens
- Live Server

## Install Python
```
brew install python@3.11
python3 --version
```

## Install Git
```
brew install git
git --version
```
